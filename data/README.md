# Data

## deta detail

```
"name": "",
"website": "",
"deleteurl": "",
"info": "",
"difficulty": "",
"keytype": "",
"priority": "",
"category": ""
```

## `name` field:
- service name

## `website` field:
- website name

### `deleteurl` field:
- `#`
- delete account url eg: `https://test.com/account-delete`

### `info` field:
- an instruction about "how to delete/deactivate an account in this service"

### `difficulty` field:

- ساده:‌ فرآیند آسان
- متوسط: به صورت چند مرحله‌ای
- سخت:‌ نیاز به ارسال پیام/تیکت به پشتیبانی
- غیرممکن: غیرقابل حذف

### `keytype` field:

- easy-label
- medium-label
- hard-label
- impossible-label
